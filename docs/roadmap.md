# Roadmap

## Milestone 0: Working Prototype (current)

- CEF host opens Pulse UI
- Sidebar with vertical tabs (tree-like groups)
- Omnibox and basic navigation actions in UI shell
- CI checks and release scaffolding

## Milestone 1: Real Browser Core

- Native navigation wiring (back/forward/reload/new tab)
- Per-tab browser state + restore (in progress)
- Session persistence and backup-based crash recovery (prototype landed)

## Milestone 2: Workspace Expansion

- File/folder items in sidebar
- Code editor surface
- Terminal panel
- Split panes

## Milestone 3: Capsules

- Capsule manifest + lifecycle
- Capability request/grant flow
- Sandboxed runtime bridge

## Milestone 4: Sync + Distribution

- Account login
- Backup/sync
- Installer + auto-update rollout
- Differential update pipeline
